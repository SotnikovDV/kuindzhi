# Архип Куинджи: Алхимик Света

Node.js приложение (Express + EJS) с интерактивной инфографикой, галереей и модальным/полноэкранным просмотром работ Архипа Куинджи.

## Структура проекта

```
kuindzhi/
├── app.js                 # Сервер (Express)
├── package.json           # Зависимости проекта
├── env.example            # Шаблон переменных окружения (скопируйте в .env)
├── views/                 # EJS шаблоны
│   └── index.ejs          # Главная страница
├── public/                # Статика
│   ├── css/style.css
│   ├── js/charts.js       # Логика модалки/галереи/AI/вьюера
│   ├── docs/kuindzhi.pdf  # PDF "Узнать больше о Куинджи"
│   ├── media/moon_night.mp3
│   └── gallary/           # Изображения картин (с префиксом года)
└── routes/                # (опционально)
```

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Настройте переменные окружения:

- Скопируйте `env.example` в `.env`
- Заполните `PERPLEXITY_API_KEY` и (опционально) `PORT`

## Запуск

### Режим разработки (с автоперезагрузкой)
```bash
npm run dev
```

### Обычный запуск
```bash
npm start
```

После запуска откройте браузер и перейдите по адресу:
```
http://localhost:3400
```

## Технологии

- **Node.js** - серверная платформа
- **Express** - веб-фреймворк
- **EJS** - шаблонизатор
- **Chart.js** - библиотека для графиков
- **Tailwind CSS** - CSS фреймворк

## Особенности

- Интерактивные графики (Chart.js)
- Галерея, сортировка по имени файла (с учётом года в префиксе)
- Модальное окно с навигацией “пред/след”, кнопкой яркости +30% и “ИИ‑искусствоведом”
- Полноэкранный просмотр в новой вкладке (в т.ч. свайпы/тап по краям на планшете/смартфоне)
- Адаптивный дизайн, скрытие скроллбаров там, где нужно
- Аудио‑пост к главной картине (Play/Pause + громче/тише)
- Ссылка “Узнать больше о Куинджи” (PDF)

## Порты

По умолчанию приложение использует порт **3400**. Для изменения порта установите переменную окружения:

```bash
PORT=8080 npm start
```
